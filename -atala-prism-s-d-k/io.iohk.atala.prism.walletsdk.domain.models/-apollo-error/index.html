sealed class ApolloError : KnownPrismError(source)

A class representing a known error in an Apollo API response.

Note: When an error occurs during an API request/response cycle, the server may return an error object in the response. This object may include an error code and an error message. If the error received conforms to the KnownPrismError, it will be classified as a known error.
